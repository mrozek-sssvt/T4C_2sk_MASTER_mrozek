CSS Vlastnosti Pozadí

Praktická ukázka různých `background` vlastností v CSS.

background-color

Nejzákladnější vlastnost, která nastavuje barvu pozadí prvku.

background-image

Nastaví obrázek jako pozadí. Standardně se opakuje (tile effect).

background-repeat: no-repeat

Zabrání opakování obrázku na pozadí.

background-size: cover

Obrázek se roztáhne tak, aby pokryl celou plochu, i za cenu oříznutí.

background-size: contain

Obrázek se zobrazí celý, přizpůsobí se velikosti boxu, ale nemusí ho vyplnit celý.

background-position

Umožňuje umístit neopakující se obrázek na konkrétní místo (např. `center top`).

background-attachment: fixed

Pozadí zůstává na místě při rolování stránky (tzv. paralaxní efekt).

Více obrázků na pozadí

Do jedné vlastnosti lze vložit více obrázků a každý stylovat zvlášť.

Zkratka: `background`

Všechny vlastnosti lze zapsat do jedné pro přehlednější kód.
