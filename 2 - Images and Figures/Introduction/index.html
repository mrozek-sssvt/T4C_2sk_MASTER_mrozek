Základní vložení obrázku

Element <img> je základ pro vkládání obrázků. Atribut alt je klíčový pro přístupnost.

Použití <figure> a <figcaption>

Pro obrázky, které doplňují obsah a mají vlastní popisek, je sémanticky správné použít <figure> a <figcaption>.

Responzivní obrázek s <picture>

Element <picture> umožňuje načíst různý obrázek pro různá zařízení (např. jiný formát nebo ořez pro mobily a desktopy).
